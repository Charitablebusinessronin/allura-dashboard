# Allura Memory Dashboard

A standalone React + Vite dashboard for the Allura Memory system. Built from the Figma Memory Dashboard design.

## Pages

- **Dreams** — Background intelligence, memory analysis, quick actions
- **Governance** — Policy compliance, evidence chains, access reviews
- **Kanban Board** — Task organization and workflow tracking

## Tech Stack

- React 19 + TypeScript
- Vite 6
- Tailwind CSS 4
- shadcn/ui compatible tokens
- React Router 7
- Lucide React icons

## Development

```bash
bun install
bun run dev
```

## Build

```bash
bun run build
```
